Conclude the presentation with the core message: structure existing materials so AI Agent can read, draft, review, and preserve human responsibility.

Create a 16:9 hand-drawn whiteboard style closing diagram. Show four connected pillars or cards:

1. "SoT" with structured data cards
2. "Context" with guide/PDF/review note cards
3. "Typst + Scripts" with a document and gear

Main message: structured data, evidence, rendering, and guardrails make AI-assisted regulatory documentation practical.
